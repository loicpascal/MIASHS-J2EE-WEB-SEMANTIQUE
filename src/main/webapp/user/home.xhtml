Bienvenue

L'application Java EE PhotoStore est une plateforme sur laquelle les internautes peuvent s'inscrire, créer leur albums et y ajouter des photos. Elle utilise les technologies du Web sémantique pour permettre aux utilisateurs d'enrichir chaque photo en y ajoutant des annotations (Ex : Qui est sur la photo ? Que représente la photo ? Où et quand la photo a-t-elle été prise ?).
Toutes ces informations sont ensuite stockées dans un graphe RDF.

À l'aide d'une ontologie (ensemble de relations sémantiques entre des éléments), il est alors possible d'effectuer des recherches complexes sur l'ensemble des photos de l'application. Exemple de recherche :

  • Retrouver toutes les photos représentant les amis d'une personne définie
  • Retrouver tous les selfies
  • Retrouver toutes les photos prises à Grenoble
  • Retrouver toutes les photos d'animaux

Le framework Apache Jena fournit une API permettant d'écrire les données dans le graphe RDF et de les extraire via le langage de requête et protocole SPARQL.
